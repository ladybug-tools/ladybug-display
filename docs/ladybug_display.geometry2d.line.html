ladybug_display.geometry2d.line module¶

A line segment that can be displayed in 2D space.

class ladybug_display.geometry2d.line.DisplayLineSegment2D(geometry, color=None, line_width=Default, line_type='Continuous')[source]¶

Bases: ladybug_display.geometry2d._base._LineCurveBase2D

A single line segment in 2D space with display properties.

Parameters

  • geometry – A ladybug-geometry LineSegment2D object.

  • color – A ladybug Color object. If None, a default black color will be used. (Default: None).

  • line_width – Number for line width in pixels (for the screen) or millimeters (in print). This can also be the Default object to indicate that the default settings of the interface should be used.

  • line_type –

    Get or set text to indicate the type of line to display. Choose from the following. (Default: “Continuous”)

    • Continuous

    • Dashed

    • Dotted

    • DashDot

    • Center

    • Border

    • Hidden
