ladybug_display.geometry3d.cylinder module¶

A cylinder that can be displayed in 3D space.

class ladybug_display.geometry3d.cylinder.DisplayCylinder(geometry, color=None, display_mode='Shaded')[source]¶

Bases: ladybug_display.geometry3d._base._SingleColorModeBase3D

A cylinder in 3D space with display properties.

Parameters

  • geometry – A ladybug-geometry Cylinder.

  • color – A ladybug Color object. If None, a default black color will be used. (Default: None).

  • display_mode –

    Text to indicate the display mode (shaded, wireframe, etc.). Choose from the following. (Default: Shaded).

    • Shaded

    • Surface

    • SurfaceWithEdges

    • Wireframe
